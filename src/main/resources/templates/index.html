Cadastrar novo Pedido

Exemplo de Requisição para cadastrar pedidos no sistema (Post)

Criar um array com os campos a seguir em seguida, enviar o ajax para o servidor processar, o retorno é exibido na sequência

Sendo necessário informar apenas os campos obrigatórios conforme especificação

numberControl número de controle

dateRegister Data de registro (campo String eviado e recebido na consulta)

name Nome

qauntity Quantidade

clientCode Código do cliente

Observação, ao enviar mais de um pedido alguns podem ser processados e outros não devido aos critérios de aceite, essas validações são tratadas na aplicação

array = [{numberControl: 1, dateRegister: '2020-05-30 09:00:00', name: 'Teste', quantity: 2, value: 10, clientCode: 15},
{numberControl: 2, dateRegister: '2020-05-30 12:00:00', name: 'Teste', quantity: 1, value: 10, clientCode: 15}];

$.ajax({
contentType: 'application/json',
data: JSON.stringify(array),
dataType: 'json',
success: function(data){
console.log(JSON.stringify(data));
},
error: function(){
console.log("Erro na requisição");
},
processData: false,
type: 'POST',
url: 'https://eicon-test-app.herokuapp.com/save'
});

Exemplo de requisição para consulta de pedidos cadastrados no sistema

clientOrderSearch: String texto de consulta

numberControl: Booleano indica se deve filtrar pelo campo número de controle

dateRegister: Booleano indica se deve filtrar pelo campo data de registro

client: Booleano indica se deve filtrar pelo campo cliente

all: Booleano indica se deve filtrar por todos os campos

Para retornar todos os resultados informar todos os registros como false

https://eicon-test-app.herokuapp.com/search?clientOrderSearch=Teste&numberControl=false&dateRegister=false&client=false&all=true
